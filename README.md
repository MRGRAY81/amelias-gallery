# amelias-gallery
A simple, family-managed art gallery site for showcasing artwork and handling commission requests.
# Amelia's Gallery Backend

## Routes
- GET /health
- POST /auth/login
- POST /commissions
- POST /upload
- GET /admin/commissions (Bearer token)

## ENV (Render)
- JWT_SECRET
- ADMIN_EMAIL
- ADMIN_PASSWORD
- FRONTEND_ORIGIN
